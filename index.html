📚 Scientific Reference

Formula: Urban Heat Island Index (UHII) based on Liu et al. (2015)

Citation: Liu et al., "Quantifying Urban Heat Island Effect in Developing Asian Cities," Remote Sensing Letters, 2015

Function: UHII = (0.42×LST) + (0.25×Urban_Built) - (0.58×NDVI) + (0.15×Water)

ℹ️ Values normalized to 0-100 scale using local temperature range (28-36°C) and NDVI variations (0.20-0.45) typical for Mumbai region.

  • Neem trees for natural cooling

  • Marigold for ground cover
  • Gulmohar for aesthetic value
